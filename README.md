# Dotfiles

My dotfiles :3

# Installing

```
mkdir -p ~/github.com/jameslaverack/
cd ~/github.com/jameslaverack
git clone https://github.com/JamesLaverack/dotfiles.git
cd dotfiles
./install-macos.sh
./tweaks-macos.sh
```

Then:
1. Launch Firefox
2. Install extensions
3. `./install-firefox-profile.sh`
